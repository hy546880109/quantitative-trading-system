### 3. 后端数据处理设计

#### 3.1 指数代码处理

**文件**: `src/data/data_feed.py`

在 `YahooFinanceDataFeed` 类的 `_convert_symbol` 方法中添加指数代码处理：

```python
def _convert_symbol(self, symbol: str) -> str:
    """转换股票代码格式"""
    # 美股指数代码（^开头）保持不变
    if symbol.startswith('^'):
        return symbol

    # 中国A股转换
    if symbol.endswith('.SH'):
        return symbol.replace('.SH', '.SS')
    elif symbol.endswith('.SZ'):
        return symbol.replace('.SZ', '.SZ')

    return symbol
```

**关键点**:
- `^GSPC`, `^IXIC`, `^DJI` 直接传递给 Yahoo Finance API
- Yahoo Finance 原生支持这些指数代码
- 无需额外转换逻辑

#### 3.2 价格格式化

**文件**: `gui.py`

添加价格格式化函数：

```python
def format_price(price: float, symbol: str) -> str:
    """
    根据股票代码格式化价格显示

    Args:
        price: 价格数值
        symbol: 股票代码

    Returns:
        格式化的价格字符串
    """
    if symbol.startswith('^'):
        # 美股指数用美元符号，千位分隔符
        return f"${price:,.2f}"
    else:
        # A股用人民币符号
        return f"¥{price:,.2f}"
```

**应用场景**:

1. **数据预览表格**:
   ```python
   st.metric("最新收盘价", format_price(st.session_state.data['close'].iloc[-1], symbol))
   ```

2. **交易记录表格**:
   ```python
   trades_df['price'] = trades_df.apply(
       lambda row: format_price(row['price'], row['symbol']), axis=1
   )
   ```

#### 3.3 错误处理优化

**文件**: `gui.py`

在数据加载部分添加针对性错误提示：

```python
try:
    if data_source == "Yahoo Finance":
        feed = DataFeed("yahoo")
        bars = feed.fetch_daily_data(
            symbol,
            start_date.strftime('%Y%m%d'),
            end_date.strftime('%Y%m%d')
        )
        st.session_state.data = bars_to_dataframe(bars)

        if st.session_state.data is not None:
            st.success(f"✅ 数据加载成功！共 {len(st.session_state.data)} 条记录")

except Exception as e:
    error_msg = str(e)

    # 美股指数友好错误提示
    if symbol.startswith('^'):
        if "network" in error_msg.lower() or "connection" in error_msg.lower():
            st.error("⚠️ 网络连接失败，请检查网络设置后重试")
        elif "No data" in error_msg or len(bars) == 0:
            st.error("⚠️ 暂无数据，可能是市场休市或日期范围无效")
        else:
            st.error("⚠️ 美股指数数据获取失败，请稍后重试")
    else:
        st.error(f"❌ 数据加载失败: {error_msg}")
```

### 4. 数据验证

#### 4.1 获取数据后的验证

```python
def validate_market_data(df: pd.DataFrame, symbol: str) -> bool:
    """
    验证市场数据完整性

    Args:
        df: 数据DataFrame
        symbol: 股票代码

    Returns:
        数据是否有效
    """
    if df is None or len(df) == 0:
        return False

    # 检查必要字段
    required_fields = ['open', 'high', 'low', 'close']
    for field in required_fields:
        if field not in df.columns:
            st.error(f"⚠️ 数据格式错误：缺少 {field} 字段")
            return False

    # 检查数据有效性
    if df[['open', 'high', 'low', 'close']].isnull().all().any():
        st.error("⚠️ 数据包含过多空值，请检查数据源")
        return False

    return True
```
